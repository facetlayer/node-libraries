# @facetlayer/claude-settings-tidier

An AI-powered CLI tool to analyze Claude Code settings files (`.claude/settings.json` and `.claude/settings.local.json`) and suggest improvements.

## Running

Analyze the settings in the current directory:

```bash
npx @facetlayer/claude-settings-tidier
```

**Authentication:** Requires the `ANTHROPIC_API_KEY` environment variable. Get your API key from [console.anthropic.com](https://console.anthropic.com/).
